授業でもただキーボードから入力するだけというような簡単なプログラムを書くことが少なくなってきたので、
より便利に標準入力が行えるInputScannerというクラスを作ってみました。

使い方はjavadocを見てください。フォルダに同封してあります。
htmlファイルの場所はこのREADME.txtファイルと同じ階層にあるjavadocの場所.txtに記述してあります。

今後さらに増えるであろう入力制限などを簡単に実現できると思います。
先生的には自作したクラスなどを授業の課題で使用しても全然OKとのことだったので使ったもん勝ちでしょう。

* まだStringの入力とintの入力しか完成していません。
  long型やchar型などこれからさらに増やし、汎用性を高めていくつもりです。

* バグやここが使いにくい、これの使い方がわかりにくいなどありましたら教えてください。改善します。
